## Getting Started

1. **Clone the Repository:**
   ```bash
   git clone https://github.com/your-username/your-project.git
   cd your-project

2. # Install backend dependencies
   cd Backend
   npm install

   # Install frontend dependencies
   cd ../Frontend
   npm install

 3. # Run the backend server
    cd Backend
    npm start

    # Run the frontend application
    cd ../Frontend
    npm start
